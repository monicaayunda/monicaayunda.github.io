Cara Membuat Bootable Windows 10 dengan Rufus

Pendahuluan

Membuat bootable USB drive adalah cara yang efektif untuk menginstal ulang Windows 10, memperbaiki sistem yang rusak, atau menjalankan alat diagnostik. Rufus adalah aplikasi gratis dan ringan yang sangat populer untuk membuat bootable USB drive dengan mudah dan cepat.

Persiapan

Sebelum memulai, pastikan Anda memiliki:

  • USB Drive: Minimal 8 GB. Pastikan data di dalamnya sudah di-backup karena akan diformat.
  • File ISO Windows 10: Unduh dari situs resmi Microsoft atau sumber tepercaya lainnya.
  • Rufus: Unduh aplikasi Rufus terbaru dari situs resminya (https://rufus.ie/).

Langkah-langkah

  1. Jalankan Rufus

    Setelah diunduh, jalankan aplikasi Rufus. Anda tidak perlu menginstalnya.

  2. Pilih USB Drive

    Pada bagian "Device", pilih USB drive yang ingin Anda gunakan.

  3. Pilih File ISO

    Pada bagian "Boot selection", pilih "Disk or ISO image". Klik tombol "SELECT" dan pilih file ISO Windows 10 yang sudah Anda unduh.

  4. Pengaturan Tambahan (Opsional)

    Beberapa pengaturan tambahan yang bisa Anda sesuaikan:

    • Partition scheme: Biasanya, Rufus akan otomatis mendeteksi yang terbaik untuk sistem Anda. Jika Anda yakin, Anda bisa memilih antara "MBR" (untuk sistem BIOS lama) atau "GPT" (untuk sistem UEFI modern).
    • Target system: Sesuaikan dengan "Partition scheme". Jika memilih MBR, pilih "BIOS or UEFI". Jika memilih GPT, pilih "UEFI (non CSM)".
    • File system: Biasanya "NTFS" adalah pilihan yang baik.
    • Cluster size: Biarkan default.

  5. Mulai Proses

    Klik tombol "START". Akan muncul peringatan bahwa semua data di USB drive akan dihapus. Klik "OK" untuk melanjutkan.

  6. Tunggu Proses Selesai

    Rufus akan mulai memformat USB drive dan menyalin file ISO ke dalamnya. Proses ini mungkin memakan waktu beberapa menit, tergantung pada kecepatan USB drive dan ukuran file ISO.

  7. Selesai

    Setelah proses selesai, status akan berubah menjadi "READY". Klik "CLOSE" untuk menutup Rufus.

Kesimpulan

Sekarang Anda memiliki bootable USB drive Windows 10 yang siap digunakan. Anda bisa menggunakannya untuk menginstal atau memperbaiki sistem operasi Windows 10 Anda.
